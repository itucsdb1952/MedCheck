This documentation is prepared for the project made by Furkan Güvenç and Emre Faruk Kolaç. Project ,named MedCheck, aims to ease the process of making an appointment form hospitals. Project have three main object: hospital, doctor and patient. We currently have all hospitals in Turkey. New hospitals can be added using admin authorities. Doctors can add themselves to the system and finally patients can register to the system and start using the site.
